API Image Search Microservice

Full stack Javascript app built with node.js, express.js, mongoDB, and Bootstrap by ECC

User stories:
    1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
    2. I can paginate through the responses by adding a ?offset=2 parameter to the URL. Each page contains 10 results. For example, offset=2 displays results 11-20.
    3. I can get a list of the most recently submitted search strings.

Example usage 1:

https://ecc-urlshortener-ms-ms.herokuapp.com/api/imagesearch/cats
